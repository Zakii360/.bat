## Features

- 🧹 System cleanup
- 🌐 Network tools
- 💻 Developer shortcuts
- ⚙️ Maintenance scripts
- 📊 System information
- 🎮 Fun demos

## Folder Overview

### Utilities
General everyday tools.

- Clear Temp Files
- Flush DNS
- Restart Explorer
- Empty Recycle Bin
- Clear Clipboard

### System

Useful Windows information.

- Battery Report
- Installed Programs
- Driver List
- Environment Variables
- Startup Folder

### Networking

Networking commands.

- Ping Test
- Traceroute
- Local/Public IP
- Renew IP
- Wi-Fi Password Viewer

### Developer

Developer productivity.

- Git Status
- Open VS Code
- Python HTTP Server
- Node Version
- Open PowerShell

### Maintenance

Windows repair scripts.

- System File Checker
- DISM Restore Health
- Disk Cleanup
- Check Disk
- Defrag

### Fun

Just because.

- Matrix
- Fake Hacker
- Rainbow Console
- ASCII Star Wars

## Usage

Download or clone:

```bash
git clone https://github.com/USERNAME/bat-toolkit.git
```

Double-click any script or run it from Command Prompt.

Some scripts require **Administrator** privileges.
